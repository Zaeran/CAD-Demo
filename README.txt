Hey folks!

This is the project folder of the CAD Viewer demo, as seen at the 2015 Avalon Air Show.

Due to licensing, the environment has been changed, however the CAD model remains, as does all functionality.

To Use:

- Run .exe
- Press 'r' to reset the camera position
- Use a two-fingered pinch to pick up pieces of the model
- Open fingers to release

- RESET button returns components to their original position
- EXPLODE button opens up model to show all components
- CREDITS button shows credits
